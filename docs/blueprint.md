# **App Name**: AuthConnect

## Core Features:

- User Login: Implement a secure login system with email and password authentication.
- User Registration: Provide a signup page for new users to create accounts with email and password.
- Dashboard Access: After successful login, redirect users to a personal dashboard page.
- Password Reset: Allow users to reset their password via email verification.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and security.
- Background color: Light gray (#F5F5F5) to provide a clean and neutral backdrop.
- Accent color: Teal (#009688) to highlight interactive elements and calls to action.
- Body and headline font: 'Inter', a grotesque-style sans-serif, for a modern and neutral look.
- Simple, intuitive form layout with clear labels and instructions.
- Use standard lock and key icons to represent security and access.
- Subtle transition animations for page navigation and form submissions.
